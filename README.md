# NgNgrxTodoList

This is a project for playing with Angular and NgRx

## Requirements

1. Angular v15+
2. NgRx v15+

## set up
1. clone the project
2. Run `npm install`

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
